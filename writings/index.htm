For any pair \(a, b\) of real numbers we have \(a \le b\) or \(b \le a\) [1].

theorem dichotomy (x : ℝ) : x > 0 ∨ x = 0 ∨ x < 0 := by

Get an \(x \in \mathbb{R}\) and prove \(x > 0 \vee x = 0 \vee x < 0\).

by_cases hx_pos : x > 0

We divide into two cases, if \(x > 0\) is true or if \(x > 0\) is false.

· exact Or.inl hx_pos

If \(x > 0\) is true then \(x > 0 \vee x = 0 \vee x < 0\).

· by_cases hx_neg : x < 0

If \(x > 0\) is false, we have a case in which \(x < 0\) is true and a case in which \(x < 0\) is false.

· exact Or.inr (Or.inr hx_neg)

\(x > 0\) is true then \(x = 0 \vee x < 0\) but \(x < 0\).

· have hx_zero : x = 0 := by
linarith

If \(x > 0\) is false and \(x < 0\) is false, that is, \(x \le 0\) and \(x \geq 0\) we have \(x = 0\).

exact Or.inr (Or.inl hx_zero)

To prove \(x > 0 \vee x = 0 \vee x < 0\) we have \(x = 0\), we choose \(x = 0 \vee x < 0\) and then \(x = 0\). The lean code here [2].
